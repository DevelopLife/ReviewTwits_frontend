## **WHY ReviewTwits ?**

---

자신이 생각하는 제품인지 알기 위해선 꼭 값을 지불해야 할까요? <br/>
통합된 리뷰를 통해 구매 고민을 줄이고, 제품 선택의 시간을 단축하고 시행착오를 줄일 수 있을까요? <br/>
ReviewTwits는 소비자가 제품을 인식하고 구매하기까지에 필요한 정보와 후기를 쉽게 제공합니다
<br/>
ReviewTwits와 현명한 소비를 즐겨보세요!
